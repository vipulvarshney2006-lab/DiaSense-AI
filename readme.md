# AI-Based Early Diabetes Risk Assessment System

## Overview

This project predicts the risk of diabetes using Machine Learning
based on publicly available healthcare data.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Streamlit

## Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- SVM

## Best Model

Random Forest

Accuracy: **96.94%**

Cross Validation Accuracy: **96.87%**

## Features

- Early diabetes risk prediction
- User-friendly interface
- Personalized health recommendations
- Educational healthcare support

## Disclaimer

This application is intended for educational purposes and is not a replacement for professional medical advice.
